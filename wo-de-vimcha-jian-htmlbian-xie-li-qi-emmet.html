我的vim插件--HTML编写利器Emmet

2014年07月21日

emmet简介

Emmet 项目原来叫 Zen Coding。由俄罗斯前端开发工程师 Sergey Chikuyonok 开发，后来在 Google Code 上释出 Zen Coding 项目，2012年更名为Emmet。 Emmet 官方支持很多软件，比如 Sublime Text、Notepad++、Dreamweaver、Eclipse、Adobe Brackets 等。Emmet.vim 并非 Emmet 官方出品，它由日本 Yasuhiro Matsumoto 开发。

emmet.vim 安装

  • 传统安装方式 你可以从两个地址下载，一是Vim 插件站点，一是Github
cd ~/.vim
unzip emmet-vim.zip
  • pathogen.vim 方式管理插件 在vim配置文件中增加：
Bundle 'mattn/emmet-vim'

然后在vim中执行：

:BundleInstall

emmet for vim 基本用法

  • emmet在vim中的默认触发快捷键是Ctrl+Y + ,
  • 展开缩写（Expand Abbreviation）模式
    在插入模式下输入html5或者! 再输入Ctrl+Y + , (注意：仍然在插入模式下)，会得到如下html文本：
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title></title>
</head>
<body>

</body>
</html>

类似的缩写还有:html:4s, html:4t, html:xs, html:xt

emmet支持的缩略词模式的语法:

  1. E、N代表HTML标签
  2. E>N 代表N是E的子元素
  3. E+N 代表N是E的同级元素
  4. E^N 代表N是E的上级元素
  5. E#id代表id属性
  6. E.class代表class属性
  7. E{foo} 标签包含文本foo
  8. E*n 产生n个相同标签
  9. foo$@m*n 代表foo为自增变量，起始值为m(默认为1)，即foom，foo(m+1)...foo(m+n-1); $同时还是占位符，如$$$表示递增值补齐为3位，不足前补零;若n前有-号，递增方向相反。
  10. [attr=foo] 代表某一个特定属性(可自定义)

示例:

缩写：nav>div>div+p^span
<nav>
    <div>
        <div></div>
        <p></p>
    </div>
    <span></span>
</nav>

缩写:div#test>span.cls
<div id="test"><span class="cls"></span></div>

缩写:div#test>span.cls1.cls2.cls3
<div id="test"><span class="cls1 cls2 cls3"></span></div>

缩写：p{foo}
<p>foo</p>

缩写:ul>li.cls$$$@-8*3
<ul>
    <li class="cls010"></li>
    <li class="cls009"></li>
    <li class="cls008"></li>
</ul>

缩写：div[attr1=foo1 attr2=foo2]
<div attr1="foo1" attr2="foo"></div>

emmet对每个html标签几乎都有缩写的支持，能极大提升书写html的效率，把更多的时间用于思考,例如:

缩写: input:t
<input id="" type="text" name="">

缩写: table+
<table>
    <tr>
        <td></td>
    </tr>
</table>

缩写：cc:ie6
<!--[if lte IE 6]>

<![endif]-->

更多缩写请参考： http://docs.emmet.io/cheat-sheet/ 目前该插件只实现了HTML部分CSS部分尚未实现

在编写HTML代码时，有时需要添加一些内容，Emmet提供了一段随机的看不懂的英文字段，作为测试数据我们就可以调用。

缩写：lorem
Lorem harum nulla cumque facilis adipisci facilis. Facere quia sint aspernatur rerum cum culpa eum totam? Reprehenderit magni beatae alias voluptates minima. Doloremque illo eligendi cumque similique libero sed ad.

缩写：lorem5
Ipsum amet culpa minus enim!

也可以和其他命令结合使用，生成测试数据(在此功能上还存在缺陷,无法支持类似p#id>lorem4.class*4这种写法)

缩写：p*4>lorem4.class
<p>Amet libero facilis maxime.</p>
<p>Elit vitae optio perspiciatis!</p>
<p>Amet maiores odio enim.</p>
<p>Amet maiores deserunt vero?</p>
  • 扩展缩写(Wrap with Abbreviation)模式
    输入如下内容:
test1
test2
test3

Shift+v进入可视模式，“行选取”上面三行内容，然后按键 Ctrl+Y+,,这时 Vim 的命令行会提示 Tags:，键入ul>li*，然后按 Enter,会得到如下文本：

<ul>
    <li>test1</li>
    <li>test2</li>
    <li>test3</li>
</ul>

快捷键介绍

  • Ctrl+Y + n 【插入模式】跳转到下一个编辑点
  • Ctrl+Y + N 【插入模式】跳转到上一个编辑点
  • Ctrl+Y + d 【插入模式】根据光标位置选中整个标签
  • Ctrl+Y + D 【插入模式】根据光标位置选中整个标签内容
  • Ctrl+Y + k 【插入模式】【普通模式】删除标签
  • Ctrl+Y + j 【插入模式】【普通模式】分割/合并标签，单标签(没有结尾标签)和双标签切换
  • Ctrl+Y + / 【注释模式】注释标签
  • Ctrl+Y + m 【可视模式】合并行
  • Ctrl+Y + a 【普通模式】【可视模式】通过URL生成a标签

输入文本：http://www.sina.com.cn，按键 Ctrl+Y + ,

<a href="http://www.sina.com.cn">新浪首页</a>

居然自动取URL的HTML文本的title值作为a标签的文本，神奇！若未联入互联网，则文本为空

  • Ctrl+Y + A 【普通模式】【可视模式】通过URL生成引用文本
  • Ctrl+Y + i 更新img标记的图片大小，仅本地图片可用
  • Ctrl+Y + c 【可视模式】把程序代码转换为html

高级用户可对快捷键键盘映射进行重新定义，:h emmet查询帮助文件，搜索 emmet-customize-keymappings

可配置项

修改缩进值

let g:user_emmet_settings = {
  \  'html' : {
  \    'indentation' : ' '
  \  },
  \}

定义不同文件类型的简写扩展, 例如：

let g:user_emmet_settings = {
\ 'php' : {
\ 'extends' : 'html',
\ 'filters' : 'c',
\ },
\ 'xml' : {
\ 'extends' : 'html',
\ },
\ 'haml' : {
\ 'extends' : 'html',
\ },
\}
let g:user_emmet_settings = {
\  'lang' : 'ja',
\  'html' : {
\    'filters' : 'html',
\    'indentation' : ' '
\  },
\  'perl' : {
\    'indentation' : '  ',
\    'aliases' : {
\      'req' : "require '|'"
\    },
\    'snippets' : {
\      'use' : "use strict\nuse warnings\n\n",
\      'w' : "warn \"${cursor}\";",
\    },
\  },
\  'php' : {
\    'extends' : 'html',
\    'filters' : 'html,c',
\  },
\  'css' : {
\    'filters' : 'fc',
\  },
\  'javascript' : {
\    'snippets' : {
\      'jq' : "$(function() {\n\t${cursor}${child}\n});",
\      'jq:each' : "$.each(arr, function(index, item)\n\t${child}\n});",
\      'fn' : "(function() {\n\t${cursor}\n})();",
\      'tm' : "setTimeout(function() {\n\t${cursor}\n}, 100);",
\    },
\  },
\ 'java' : {
\  'indentation' : '    ',
\  'snippets' : {
\   'main': "public static void main(String[] args) {\n\t|\n}",
\   'println': "System.out.println(\"|\");",
\   'class': "public class | {\n}\n",
\  },
\ },
\}

修改默认触发快捷键：

let g:user_emmet_expandabbr_key = '<C-Y>,'

如果完成tags要使用omnifunc方式，添加如下配置，关于omnifunc请通过h omnifunc查询帮助

let g:use_emmet_complete_tag = 1

让emmet只对html,css生效

let g:user_emmet_install_global = 0
autocmd FileType html,css EmmetInstall

修改emmet插件的使用模式范围,n代表普通模式，i代表插入模式，v代表可视模式

let g:user_emmet_mode='n'   " 只保留普通模式下的功能
let g:user_emmet_mode='iv'  " 只保留插入模式和可视模式下的功能
let g:user_emmet_mode='a'   " 支持所有模式

自定义snippets
由于这个功能需要依赖插件作者的另一个插件web-api,而web-api插件需要依赖curl或者wget,这样使得跨平台使用显得非常繁琐，而这个自定义功能可通过之前介绍的xxx插件来完成。 如果依然对此有兴趣，可查阅插件帮助，搜索Adding custom snippets,并参考

http://docs.emmet.io/customization/snippets/

小结

emmet功能异常的强大，对于经常手写HTML代码的前端攻城狮来说简直是必备神器，对于emmet for vim 唯一遗憾的是，作者还没开发css扩展的部分，比起Sublime Text2版本的插件，稍有逊色。但是从github上状态看，该项目还在不断的开发中，建议使用者时不时的更新，或许会有意外的惊喜。
